Krishi Mitra is a web-based agricultural assistance platform designed to
support farmers by providing crop guidance, weather updates, agricultural
news, and essential farming resources in one place. The platform aims to
bridge the information gap and promote informed decision-making in
agriculture.

## Features
- Crop advisory and crop guide for farmers
- Real-time weather information
- Latest agricultural news updates
- Fertilizer and services information
- User authentication using Firebase
- Simple, responsive, and user-friendly interface

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend/Services:** Firebase (Authentication & Hosting)
- **Tools:** Git, GitHub, VS Code
